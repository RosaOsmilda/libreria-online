Portal web para consultar el catálogo de libros y autores de una librería online.

🌐 **Sitio publicado:** [libreria-rossa.infinityfreeapp.com](http://libreria-rossa.infinityfreeapp.com)

## 🛠️ Tecnologías
- PHP 8 + PDO
- MySQL
- Bootstrap 5
- JavaScript (buscador en tiempo real)
- HTML5 + CSS3

## 🚀 Funcionalidades
- Catálogo de libros con buscador en tiempo real
- Listado de autores con tarjetas informativas
- Formulario de contacto que guarda en base de datos
- Página de inicio con estadísticas (COUNT, sizeof)
